#  Sistema de Folha de Pagamento

Projeto desenvolvido em Java para realizar o gerenciamento e cálculo da folha de pagamento de funcionários, aplicando conceitos de Programação Orientada a Objetos (POO) e boas práticas de desenvolvimento.

##  Funcionalidades

- Cadastro de funcionários
- Registro de informações salariais
- Cálculo de salário bruto
- Cálculo de descontos
- Cálculo de salário líquido
- Exibição de dados dos funcionários
- Geração de relatório da folha de pagamento

##  Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)

## Conceitos Aplicados

- Classes e Objetos
- Encapsulamento
- Herança 
- Polimorfismo 
- Métodos e Construtores
